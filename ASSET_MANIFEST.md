# ASSET_MANIFEST — عالم حكيم
مرجع المواصفات الفنية النهائي لكل أصل يحتاجه التطبيق. حاليًّا التطبيق
يعمل بكامل طاقته ببدائل متدرّجة (Resilient Assets Pattern)؛ تُستبدل
بالأصول الاحترافية فور جاهزيتها دون تعديل الكود (تُسجّل في config/assets.ts).

نمط التصميم العام: Ultra-Premium Playful — كرتوني ثلاثي الأبعاد ناعم،
ألوان دافئة (برتقالي #FF9F1C / #F57C00)، حوافّ مستديرة، إضاءة لطيفة.

## آلية الاستبدال (عند الجاهزية)
1. ضع الملفّات في مجلّداتها أعلاه داخل assets/.
2. في config/assets.ts، أضف كل ملفّ إلى خريطة REGISTERED عبر require.
3. لأيقونة/سبلاش التطبيق: حدّث المسارات في app.json.
4. أعد البناء (تغيير أصول native = بناء كامل، لا OTA).
